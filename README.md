# Basic To-Do

Eine einfache, elegante To-Do Webseite zum Verwalten von Aufgaben und Sachverhalten.

## 🚀 Live Demo

https://damagevindi.github.io/basic-to-do

## ✨ Features

- 📝 **Sachverhalte eingeben** - Einfaches Hinzufügen neuer Aufgaben
- ✅ **Aufgaben erledigen** - Checkbox zum Markieren als erledigt
- 🗑️ **Löschen** - Einzelne Aufgaben oder alle erledigten auf einmal löschen
- 📊 **Statistiken** - Übersicht über Gesamt, Offen und Erledigt
- 💾 **LocalStorage** - Daten bleiben beim Browser-Refresh erhalten
- 📱 **Responsive** - Funktioniert auf Desktop und Mobile

## 🛠️ Technologien

- HTML5
- CSS3 (Gradienten, Animationen, Flexbox)
- Vanilla JavaScript (ES6+ Klassen)
- LocalStorage API
